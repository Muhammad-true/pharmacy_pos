# Аптека Хушдил - POS Система

Профессиональная система управления точкой продаж (POS) для аптек с поддержкой нескольких касс, управления складом, клиентской базой и полной статистикой.

## 🚀 Возможности

### Основные функции
- ✅ **Управление продажами** - Быстрое оформление чеков с поддержкой скидок и бонусов
- ✅ **Многокассовая система** - Поддержка нескольких касс с общей базой данных
- ✅ **Управление складом** - Учет товаров, остатков, движения товаров
- ✅ **Клиентская база** - Управление клиентами, бонусная система, история покупок
- ✅ **Смены кассиров** - Отслеживание смен, выручки, статистики по кассирам
- ✅ **История продаж** - Полная история чеков с фильтрацией по датам
- ✅ **Многоязычность** - Поддержка 6 языков (Русский, Узбекский, Английский, Таджикский, Казахский, Киргизский)
- ✅ **Настройки** - Кастомизация темы, цветов, названия аптеки
- ✅ **Дашборд** - Полная статистика продаж, выручки, популярных товаров

### Роли пользователей
- **Администратор** - Полный доступ ко всем функциям
- **Кассир** - Оформление продаж, работа со сменами
- **Склад** - Управление товарами и остатками

## 🔧 Установка

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd dorukhonai_man
```

### 2. Установка зависимостей
```bash
flutter pub get
```

### 3. Настройка базы данных

Создайте файл `.env` в корне проекта:

```env
# Путь к базе данных
# Для локальной БД:
DATABASE_PATH=./data/dorukhonai.db

# Для общей БД на сервере (для нескольких касс):
# DATABASE_PATH=\\SERVER\Shared\PharmacyDB\dorukhonai.db

# Настройки приложения
APP_NAME=Аптека Хушдил - POS
APP_VERSION=1.0.0
LOG_LEVEL=info
CASHIER_NAME=Касса 1
ENVIRONMENT=production
```

### 4. Генерация кода
```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

### 5. Запуск приложения
```bash
flutter run -d windows
```

## 📖 Настройка для нескольких касс

Для работы нескольких касс с одной базой данных:

1. **Разместите БД на сервере:**
   - Создайте общую папку: `\\SERVER\Shared\PharmacyDB\`
   - Скопируйте туда файл `dorukhonai.db`
   - Настройте права доступа (чтение/запись для всех касс)

2. **Настройте каждую кассу:**
   - В файле `.env` укажите путь к общей БД:
   ```env
   DATABASE_PATH=\\SERVER\Shared\PharmacyDB\dorukhonai.db
   CASHIER_NAME=Касса 1  # Уникальное имя для каждой кассы
   ```

3. **Важно:**
   - Используйте стабильное сетевое соединение
   - Рекомендуется максимум 5-10 одновременных подключений
   - Регулярно делайте резервные копии БД

Подробнее см. [DATABASE_SETUP.md](DATABASE_SETUP.md)
